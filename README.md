# bfun-interpreter
A simple [brainfuck](https://esolangs.org/wiki/Brainfuck) interpreter with silent mode and debug mode for runtime visualization.

#### Make

```bash
make bfun
```

#### Test

For silent mode:

~~~bash
./bfun helloworld.bf -s
~~~

Should output:

~~~
Hello World!
~~~

For debug mode:

~~~bash
./bfun helloworld.bf -d
~~~

Have fun.
